== Description ==

This plugin adds a customizable Follow Button at bottom right corner of your Jetpack-powered Wordpress.ORG site.

The plugins integrates with the built in subscription manager of the Jetpack Plugin to collect the email of the subscriber and handle distribution of new posts via email.

<strong>NOTE: This plugin requires the Jetpack plugin to be installed & activated</strong>

For further information visit the plugin website. 

== Installation ==

1. Install and Activate the "Jetpack" Plugin
2. Connect Jetpack to WP.COM Site 
3. Install and Activate the "Better Follow Button for Jetpack" Plugin

== Frequently Asked Questions ==

= 1. What does this plugin do? =
This plugin adds a floating follow button to Jetpack Powered sites that allows your readers to subscribe to your posts via email.
